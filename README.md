.emacs.d
===

My emacs configures


Requirement
---

### Minimum requirements
- `git`
- `emacs` >= 24.4
- `cask`

### Utilities
- `ag` (The Silver Searcher)

### Language support
#### Ruby
- `rubocop`
- `pry`

#### Web/Node.js
- `tern`
- `eslint`


Setup
---

```bash
$ cd ~
$ rm -rf .emacs.d
$ git clone https://github.com/yadex205/.emacs.d.git
$ cd .emacs.d
$ cask install
```
